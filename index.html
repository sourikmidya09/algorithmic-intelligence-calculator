// CS FUNDAMENTALS

Computational Thinking
Demonstration

Core concepts of computer science and algorithmic thinking visualized and explained.

🔁
Recursive Problem Solving
Recursion breaks complex problems into self-similar subproblems. This platform demonstrates base cases, recursive calls, and call stack depth through live Fibonacci trees and step-by-step traces.
T(n) = T(n-1) + T(n-2)
🧮
Matrix Mathematics
Determinants form the backbone of linear algebra — from solving systems of equations to 3D transformations. Gaussian elimination reduces any NxN matrix through row operations with full working shown.
det(A) via cofactor expansion
⚙️
Algorithm Design
Sorting algorithms illustrate the tradeoffs between simplicity (Bubble O(n²)) and efficiency (Merge O(n log n)). Visual bars animate comparisons, swaps, and pivots in real time.
divide & conquer · in-place
⚡
Optimization Techniques
Dynamic programming memoizes overlapping subproblems. The Fibonacci DP demo reduces 2ⁿ calls to O(n) by caching intermediate results — a foundational optimization concept in CS.
memoization · tabulation
🗃️
Data Structures Foundations
Arrays underlie sorting visualizations; stacks model recursive call depth; trees represent Fibonacci call graphs. Each feature intentionally demonstrates a core data structure in action.
arrays · stacks · trees
📊
Computational Complexity
Big-O notation quantifies how algorithms scale. The interactive chart plots six complexity classes simultaneously, making the catastrophic growth of exponential algorithms immediately intuitive.
O(1) → O(2ⁿ)

// PROJECT SIGNIFICANCE

Why This Project Matters

A technical showcase demonstrating core computer science principles through interactive implementation.

01

Computational Reasoning

Every module is grounded in mathematical proof — from Gaussian elimination to the Euclidean GCD algorithm, solutions are derived, not approximated.

02

Mathematical Problem Solving

Exposes the student's comfort with linear algebra, number theory, combinatorics (Pascal's Triangle), and modular arithmetic through working implementations.

03

Algorithmic Thinking

Each feature required choosing the right algorithm for the task — Sieve of Eratosthenes for primes, divide-and-conquer for merge sort, backtracking for recursion trees.

04

Recursive Logic

Fibonacci's call tree and the matrix cofactor recursion both demonstrate deep understanding of base cases, recursive depth, and stack management.

05

Software Engineering Principles

Built with vanilla ES6, modular functions, no external dependencies — demonstrating understanding of DOM, Canvas API, state management, and animation loops.

06

Applied Data Structures

Sorting visualizer directly demonstrates array manipulation; recursion tree uses a node/edge tree structure built from scratch to visualize call graphs.
