## ⚡ Core Engineering Features

### 1. Linear Algebra: Matrix Determinant Engine
* **Multi-Dimensional Support:** Computes determinants for $2\times2$, $3\times3$, and higher-order $N\timesN$ matrices.
* **Algorithmic Depth:** Implements a recursive Laplace (cofactor) expansion algorithm.
* **Step-by-Step Trace:** Dynamically breaks down signs, coefficients, and minor matrices in real-time, pulling mathematical concepts out of abstract code blocks.

### 2. Functional Induction: Recursive Fibonacci Architecture
* **Stack Visualization:** Generates an interactive, canvas-rendered execution tree mapped to user inputs ($N$).
* **Optimization Paradigm:** Compares unoptimized tree recursion ($O(2^n)$ time complexity) against a space-efficient Dynamic Programming array solution ($O(n)$ time complexity).
* **Telemetry Monitoring:** Captures microsecond-accurate runtimes to illustrate the real-world performance cost of stack-frame overhead.

### 3. Asymptotic Curve Graphing Engine
* **Interactive Charting:** Plots baseline Big-O complexity bounds ($O(1)$, $O(\log n)$, $O(n)$, $O(n \log n)$, $O(n^2)$, and $O(2^n)$) on a dynamic HTML5 Canvas.
* **Visual Scaling:** Illustrates how computational workload trends grow toward infinity as data input limits scale ($N \to \infty$).

### 4. Dynamic Abstract Data Sorting Center
* **Algorithms Implemented:** Bubble Sort, Quick Sort (Partition Model), and Merge Sort (Divide & Conquer).
* **Frame-Throttled Render:** Custom asynchronous sleep systems render the array mutations step-by-step, visually highlighting target elements, pivots, and indices during execution.

### 5. Theoretical Discrete Mathematics Playground
* Includes multi-purpose mathematical engines: Prime Number Generation via an optimized Sieve of Eratosthenes, Euclidean GCD/LCM calculators, structural Combinatorial Factorial logic, Pascal's Binomial Triangle map, and an integer-to-binary Radix Transformer.
